[![License](https://img.shields.io/badge/license-Apache--2.0-blue.svg)](LICENSE)

## History

This repository hosted the development of Dependency-Track v5 under the codename **Hyades** while the major-version rewrite was in progress.
At v5.0.0 GA, its contents were merged into the canonical [DependencyTrack/frontend](https://github.com/DependencyTrack/frontend)
repository, replacing the v4 codebase on `main`. v4 continues in maintenance mode on the `4.14.x` branch until end-of-life.

This archive remains available for reference. All Git history from this repository has been preserved in the new home.

## Documentation

User-facing documentation is rendered at [dependencytrack.github.io/docs/next](https://dependencytrack.github.io/docs/next/) and maintained in the [docs](https://github.com/DependencyTrack/docs) repository.
